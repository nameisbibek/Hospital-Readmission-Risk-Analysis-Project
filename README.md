# Hospital Readmission Risk Analysis (SQL Project)

  This project focuses on analyzing hospital readmission data using Microsoft SQL Server to identify patterns, risk factors, and key healthcare performance indicators. The dataset contains patient information such as age group, hospital stay duration, laboratory procedures, medications, medical specialties, diabetes treatment, and readmission status.

  During the project, I performed data quality checks by identifying missing values, validating data consistency, and checking for duplicate records. I then conducted exploratory data analysis (EDA) to examine patient demographics, medical specialties, diabetes medication usage, hospital stay duration, and readmission trends across different age groups.

To gain deeper insights, I performed readmission risk analysis, including:
- Readmission rates by medical specialty
- Impact of diabetes medication on readmission
- Hospital utilization analysis using emergency, inpatient, and outpatient visits
- High-risk patient identification using custom risk flags
- KPI reporting for hospital performance metrics

Additionally, I applied advanced SQL concepts such as:
- Common Table Expressions (CTEs) for organizing complex queries
- Window Functions for ranking, partitioning, and comparative analysis
- CASE statements for feature engineering and risk classification
- Views to create reusable analytical datasets and executive dashboards
